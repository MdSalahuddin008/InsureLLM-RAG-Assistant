# InsureLLM RAG Assistant 🏢🔍

A retrieval-augmented generation (RAG) assistant for **Insurellm**, a fictional insurance technology company. Employees can ask natural-language questions and get answers grounded in the company's own knowledge base — products, contracts, and employee records.

The project includes a basic RAG pipeline and a more advanced ("pro") implementation with semantic chunking, query rewriting, and LLM re-ranking, plus an evaluation suite to compare the two.

👉 Full project, setup instructions, and file breakdown are in [`insurellm-rag-assistant/`](./insurellm-rag-assistant).
